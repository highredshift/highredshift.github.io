JWST Lens Simulations: Smallest Einstein Rings

Based on: E. Silver, R. Wang, X. Huang et al. (2025)

We study how small an Einstein radius (\(\theta_E \)) can be detected with JWST, using realistic simulated background source galaxies. We find that we can detect systems with $\theta_E$ down to the diffraction limit of JWST, or ~0.03″.

Key Results

  • We simulated strong lens systems with both conventional-sized $\theta_E$ ($\gtrsim 0.5''$) and very small $\theta_E$ systems ("dimple lenses") down to 0.03″, the diffraction limit of JWST NIRCam.
  • Realistic sources — for the first time — drawn from the VELA simulations.
  • We usd a custom ResNet to find both conventional strong lenses and these small "dimple lenses"
  • We applied a U-Net to pinpoint the location of the small $\theta_E$ systems.
  • For both the ResNet and U-Net, we set high thresholds set to control false positives.
  • We determined that we can indeed detect small $\theta_E$ systems with JWST down to ~0.03″.
  • Our codes can be found at:
    • JWST Lens Simulations (JLS) — code for simulating compact Einstein rings and arcs.
    • ResNet + U-Net Pipeline (RUN) — classification and detection pipeline for small-θE lenses.

Implications

  • Guides JWST targeting for compact/dwarf-galaxy lenses.
  • Informs time allocation and follow-up strategies for systems identified in wide-area searches.
  • Provides baselines for future Roman synergy and small-θE population studies.
