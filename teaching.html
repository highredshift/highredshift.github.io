AI Series

PHYS 301 — Intro Scientific Computation

Foundations of scientific computing (Python/NumPy), data handling, visualization, and an intro to ML workflows for physical sciences.

  • support Vector Machine

PHYS 302 — Sci. Comp. & Machine Learning

Scientific ML: feature engineering, model selection, cross-validation, and applications to astronomical datasets.

  • Neural Network Basics

PHYS 303 — Bayesian/Deep Learning in Sci.

Bayesian inference (priors, posteriors, MCMC/HMC), uncertainty quantification, and modern deep learning for scientific discovery.

  • Bayes Theorem
  • Attention in Transformers
  • Build Your Own GPT
